# Portfolio_Project_Supermarket

## Overview
Analyzed retail sales data from 2014–2017 using Excel to identify trends in profitability, regional performance, and seasonality.

## Tools Used
- Excel (Pivot Tables, Pivot Charts)

## Key Insights
- West and East regions lead in profitability
- Technology drives the majority of profit
- Profit peaks in Q4 each year
